evenZoom

evenZoom is a jQuery plugin, that adds zoom functionality to images

Example

Photo by sea turtle

Version

1.0

Files include

1. include jQuery

<script type="text/javascript" src="http://code.jquery.com/jquery-1.6.3.min.js"></script>

2. include evenZoom.css

<link rel="stylesheet" type="text/css" href="evenZoom.css" />

3. include evenZoom.js

<script type="text/javascript" src="evenZoom.js"></script>

Basic Usage

Add data-zoomed attribute to #example

            <div id="example" data-zoomed="img/img-zoomed.jpg">
    <img src="img/img-original.jpg" alt="Example" />
</div>

Initialize evenZoom on #example selector

            $(function() {
    $("#example").evenZoom();
});

You can customize lens style in evenZoom.css

            .evenZoom .evenZoomLens {
    background-color: #fff;
    box-shadow: -5px -5px 30px rgba(255, 255, 255, .7) inset,
                5px 5px 20px rgba(0, 0, 0, .3) inset,
                10px 10px 25px rgba(0, 0, 0, .1);
    width: 180px;
    height: 180px;
    border-radius: 100%;
    -webkit-transition: opacity .2s;
    -moz-transition: opacity .2s;
    -o-transition: opacity .2s;
    transition: opacity .2s;
}

Properties

1. lensPinningDistance

Number of pixels lens is getting pinned when leaving range of minimal or maximal position

Default: 
20
            $(function() {
    $("#example").evenZoom({
        lensPinningDistance: 20
    });
});
